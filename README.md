## 📥 How to Download & Install

We distribute our software directly through GitHub Releases to ensure you always get the secure, virus-free, and latest version.

1.  Navigate to the **[Releases Section](https://github.com/latest)** on the right sidebar.
2.  Choose the version compatible with your device:
    *   **Windows:** Download the `.exe` file (e.g., `SmartSchool-Setup-v1.0.exe`).
    *   **Android:** Download the `.apk` file (e.g., `ProfessApp-v2.1.apk`).
    *   **macOS:** Download the `.dmg` file.
3.  Run the installer and follow the on-screen instructions.

> **Note:** For Android `.apk` installations, you may need to enable *"Install from Unknown Sources"* if downloading outside the Play Store.
